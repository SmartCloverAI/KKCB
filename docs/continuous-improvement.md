# Continuous improvement roadmap

## Phase 1: launch-ready trust layer

- finalize domain and public contact channels
- replace placeholder internal links with live donation and inquiry flows
- add legal pages: privacy, cookies, terms
- align the public copy with board/legal review

## Phase 2: conversion layer

- add volunteer inquiry form
- add donation page and recurring giving flow
- add partner inquiry path for hospitals, universities, and sponsors
- add newsletter capture

## Phase 3: research layer

- publish pilot study framework
- add advisory board / partner section
- publish downloadable one-pager for EU grant conversations
- add outcomes or case study summaries when available

## Phase 4: publishing cadence

- publish one Romanian and one English article per month minimum
- add author bios and article categories
- create recurring formats: bedside stories, partner notes, funding explainers

## Phase 5: media system

- add captions, consent status, and usage notes for every archived image
- create smaller derivatives and social share crops
- add a simple gallery page if the archive grows
